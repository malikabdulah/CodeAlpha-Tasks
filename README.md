# CodeAlpha C++ Programming Internship 💻

Welcome to my repository! This space contains the projects I developed during my **C++ Programming Internship** at **CodeAlpha** (February 2026). 

This repository highlights my practical application of C++ fundamentals, specifically focusing on mathematical logic, terminal-based user interfaces, and secure file handling.

## 🛠️ Tech Stack
* **Language:** C++
* **Core Concepts Applied:** Object-Oriented Programming (OOP), File I/O, Control Structures, and Mathematical Logic.

## 🚀 Internship Tasks

### 1. CGPA Calculator
A terminal-based academic tool designed to calculate a student's Semester Grade Point Average (SGPA) and Cumulative Grade Point Average (CGPA).
* **Features:** * Allows users to input the number of courses, credit hours, and grades obtained.
  * Dynamically calculates the total grade points and overall GPA.
* **Highlight:** Demonstrates strong logical flow, loops, and data validation in C++.

### 2. Login and Registration System
A secure user authentication system built entirely in the terminal.
* **Features:** * Users can register a new account by creating a username and password.
  * Existing users can log in by verifying their credentials.
* **Highlight:** Implements secure C++ file handling (`fstream`) to store, read, and verify user credentials locally. Data persists even after the program terminates.
